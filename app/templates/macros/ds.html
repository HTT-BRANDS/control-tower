{# ds-template Jinja Macros — canonical Azure Governance Platform UI primitives. Ports the React `@/components/ds-template/*` library from master-hub-infra into Jinja2 macros so server-rendered HTMX pages get the same look + feel as the master-hub React dashboards. Architecture: per ADR-0005 Layer 1: design-tokens.css (CSS custom properties) Layer 2: tailwind-output.css (generated utility classes) Layer 3: design-utilities.css (brand helpers + components) Layer 4: ds.html (THIS FILE — composed primitives) Naming convention: every macro is `ds_` so call sites read like `{{ ds_button("Save") }}` and so we can deprecate the older `brand_*` macros in `ui.html` cleanly.
